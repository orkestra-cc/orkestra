# Addons

The pages in this section are **auto-synced** from each addon's public repo `README.md` at build time (and nightly via cron). They are not edited here directly — to fix a typo or expand content, PR the source repo.

If you don't see an addon listed yet, the sync script may not have run. Re-trigger from `.github/workflows/sync-nightly.yml` or run `npm run sync:modules` locally.
